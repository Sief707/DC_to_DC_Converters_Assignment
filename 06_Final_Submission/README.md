# \# DC-to-DC Buck Converter Assignment

# 

# \## Project Overview

# 

# This project presents the design and simulation of an open-loop DC-DC buck converter using MATLAB/Simulink and Simscape Electrical Specialized Power Systems.

# 

# The converter was designed to:

# \- step down a 24 V DC input,

# \- produce approximately 7 V output,

# \- and supply 3 A output current.

# 

# The project includes:

# \- theoretical converter design,

# \- open-loop Simulink implementation,

# \- CCM verification,

# \- critical conduction analysis,

# \- DCM analysis,

# \- waveform export and comparison,

# \- and organized engineering documentation.

# 

# \---

# 

# \## Main Specifications

# 

# | Parameter | Value |

# |---|---|

# | Input Voltage | 24 V |

# | Output Voltage | 7 V |

# | Output Current | 3 A |

# | Switching Frequency | 10 kHz |

# | Duty Cycle | 29.2 % |

# | Inductance | 103.25 µH |

# | Capacitance | 857.14 µF |

# \## Operating Modes Investigated

# 

# The converter operation was analyzed under:

# \- Continuous Conduction Mode (CCM)

# \- Critical Conduction Mode

# \- Discontinuous Conduction Mode (DCM)

# 

# Mode transition was demonstrated using:

# \- inductance variation,

# \- and load resistance variation.
